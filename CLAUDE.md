# Go Go Gaia Website Documentation

## Overview
This is the marketing website for Go Go Gaia, an all-in-one women's health tracking app developed by Holland Neurotech Inc. The website is hosted on GitHub Pages at www.go-go-gaia.com.

## Project Structure
```
gaia-health.github.io/
├── index.html              # Main landing page
├── styles.css              # Custom styling
├── script.js              # JavaScript functionality
├── CNAME                  # Custom domain configuration
├── robots.txt             # Search engine directives
├── sitemap.xml           # Site structure for SEO
├── Logo1.png             # Company logo
├── blog/                 # Blog content
│   ├── index.html        # Blog landing page
│   ├── cycle-syncing-guide.html
│   ├── habit-tracking-success.html
│   ├── mood-tracking-benefits.html
│   └── pregnancy-wellness-tips.html
└── images/               # App screenshots and assets
    └── [various app screenshots]
```

## Key Features
- **Responsive Design**: Built with Bootstrap 5.3.2 for mobile-first responsiveness
- **Interactive Elements**: 
  - Image carousel showcasing app screenshots
  - Dynamic text animation ("Understand your health/cycle/body/mood")
  - Testimonial carousel
  - FAQ accordion
- **SEO Optimized**: 
  - Comprehensive meta tags and Open Graph data
  - Structured data (Schema.org) for app discovery
  - Sitemap for search engines
- **Analytics**: Google Analytics integration (G-ZEM6KB1QWF)
- **Blog Section**: Educational content about women's health topics

## Technologies Used

- Animations are triggered on scroll using Animate.css
- The dynamic word animation cycles through different health aspects
- All external links open in new tabs for better user experience
- The site includes both free and premium app features messaging
